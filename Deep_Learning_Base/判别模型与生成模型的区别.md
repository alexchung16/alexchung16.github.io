# 判别模型与生成模型的区别

## 判别模型 

判别模型（Discriminative model ）直接学习条件概率分布（condition probability  distribution） $p(y|x)$

## 生成模型

生成模型（Generative model）学习联合概率分布（joint probability  distribution）$p(x,y)$， 然后再使用贝叶斯理论预测条件概率。
